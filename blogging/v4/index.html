luke.is v4 released

…kinda. So I’ve just released version 4 of my personal site. It’s far from complete (more on that in a bit), but it’s a decent enough starting point.

The site is hosted on github, generated using Jekyll, written (mostly) in Markdown, CSS is preprocessed with SASS, and everything’s tied together nicely with Grunt. The Grunt usage in this project alone deserves a post of it’s own, so I’ll try to write that soon.

My TODO list still has a lot on it — priorities are 404s, removing some lazy mystery meat navigation, and then getting as much content in as possible. But for now, I’m just glad I don’t have to deal with that awful Expression Engine site anymore.
